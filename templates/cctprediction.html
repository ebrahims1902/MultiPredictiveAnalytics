CreditCard Transaction Prediction

This is an credit card detection, in this project build a machine learning system in python that can detect whether the credit card transaction legit or fraud.


Work flow:
Credit card data ➡️ data preprocessing ➡️ data analysis ➡️ train test split ➡️ logistic regression model ➡️ evaluation

This dataset is highly unbalanced. We have two labels (0 - normal transaction, 1 - fraud transaction) We had separated all normal transaction and legit transaction from this data frame with two variables.


done a statistical measure of data for legit and fraud in separate from. And then we compare the value of two transaction it is used for machine learning for identify whether transaction are legit or fraud.

We had split the data into train and test. And then we created a two variables x (feature) and y (class). We already import the sklearn. Model for train-test-split library for splitting train and test the data.
And we mentioned 4 variables x-train, x-test, y-train, y-test. 20 % of data used for test the data and remaining 80% of data used for train in randomly. And found size of for 4 variables.


We used logistics regression model. Generally logistics regression model are used for binary classification models. And we already import sklearn logistics regression library. We created the model is Logistic regression.
We train a data with help of Logistic regression model, for train a data we need to mention the function ".fit" is used for fit the data into Logistic regression model.

we need to find a performance for our model evaluation .and model evaluation based on accuracy score.
we need to find accuracy score for train data and test data separately with the help of model.

In this project we created a machine learning system in python . that can detect whether the credit card transaction Legit or fraud using Logistic regression model.


And then we split the train data and test data randomly and it provided the best accuracy score more than 93%.
